# CODSOFT Data Science Internship

This repository contains the projects completed during the CODSOFT Data Science Internship. The objective of these projects was to understand the basics of data science, machine learning, data pre-processing, data visualization, and prediction using Python.

## Projects Included

### 1. Sales Prediction Using Python

* Developed a machine learning model using Linear Regression.
* Predicted product sales based on TV, Radio, and Newspaper advertising expenditure.
* Performed data preprocessing, data visualization, model training, and model evaluation.
* Achieved a high R² score for prediction accuracy.

### 2. Iris Flower Classification

* Built a classification model using the K-Nearest Neighbors (KNN) algorithm.
* Classified iris flowers into Setosa, Versicolor, and Virginica species.
* Used sepal length, sepal width, petal length, and petal width as input features.
* Created graphs and checked the accuracy of the model.

### 3. Titanic Survival Prediction

* Created a Titanic survival prediction model using Random Forest Classifier.
* Predicted whether a passenger would survive based on passenger details.
* Used features such as age, gender, passenger class, fare, siblings/spouse, and embarkation point.
* Handled missing values and converted categorical data into numerical form.
